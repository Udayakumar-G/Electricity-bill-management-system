The Electricity Bill Management System is a web-based application 
designed to allow electricity boards to manage customer registrations, 
meter readings, billing, invoicing, and payment collection efficiently. The 
platform provides a clean and responsive interface where administrative 
staff can manage customer accounts and billing records, while registered 
customers can view their bills and make payments online through a secure 
customer portal. It uses HTML, CSS, and JavaScript for the frontend, with 
Node.js and Express for the backend, and MySQL for data management. 
Each customer, account, billing record, invoice, and payment is stored in a 
normalized relational database to ensure data integrity and reduce 
redundancy. Customer authentication is secured using email and password 
credentials stored in the customer authentication table. The admin module 
helps monitor and update customer data, billing units, and tariff records. 
CRUD operations are implemented throughout to enable dynamic content 
handling across all modules. The system includes a payment_history table 
that records every bill payment made through the portal. The project 
highlights the role of DBMS in real-time utility billing platforms. It 
provides an efficient, user-friendly environment for electricity bill 
management and digital payment.
